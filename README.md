# Portafolio — Juan Cuero

Portafolio personal desarrollado desde cero con HTML, CSS y JavaScript vanilla.

**Sitio en vivo:** https://jr-juan.github.io/Portafolio

---

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 — Flexbox, variables personalizadas, animaciones, media queries
- JavaScript — Canvas API, IntersectionObserver, manipulación del DOM
- Firebase Firestore — contador de mensajes en tiempo real
- Bootstrap Icons
- Google Fonts — Inter, Fira Code

## Características

- Animación de partículas en el fondo con Canvas
- Transiciones de secciones al hacer scroll
- Tarjetas de proyectos con enlaces a repositorios
- Sección de contacto con enlaces directos
- Descarga de hoja de vida
- Navegación fija
- Diseño responsive para móvil
